# UnityAgentVerse
Unity 3D 场景，实时可视化 AI Agent 的运行状态——Agent 们在虚拟空间里"工作"，你能看到它们的思考、通信、工具调用。后端接阿里云函数计算，前端 Unity 做 3D 呈现。

AgentVerse 核心概念


    ┌─────────────────────────────────────────────┐
    │                Unity 3D 场景                  │
    │  ┌─────┐  消息飞过去   ┌─────┐               │
    │  │ Agent│──────────→  │ Agent│               │
    │  │  🤖  │  ← 工具调用   │  🛠️  │               │
    │  └─────┘              └─────┘               │
    │     │ 状态粒子环绕        │                    │
    │     ▼                   ▼                    │
    │  "思考中..."         "调用 API..."             │
    ├─────────────────────────────────────────────┤
    │         WebSocket / HTTP                     │
    ├─────────────────────────────────────────────┤
    │     阿里云 FC (Agent 运行时)                   │
    │     LLM API (通义千问 / DeepSeek)              │
    │     OSS (任务存储)                             │
    └─────────────────────────────────────────────┘


    - 每个 Agent 在 3D 空间里是一个"小黑"风格的角色实体
    - Agent 思考时头顶冒泡、调用工具时飞粒子到目标、完成任务时放烟花
    - 你可以从 Hermes 聊天窗口派任务给 Agent，Unity 实时呈现执行过程
